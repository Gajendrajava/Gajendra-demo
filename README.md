# Gajendra-demo
💼 Employee Management System (Java Swing Project)
📌 Project Description

This is a Java-based Employee Management System developed using Swing & AWT for GUI.
It helps users to manage employee data easily with features like login, adding employees, viewing records, and attendance management.

This project is beginner-friendly and useful for understanding:

Java GUI Development
Event Handling
Basic CRUD Operations
🚀 Features
🔐 Login & Signup System
➕ Add Employee Details
👨‍💼 View Employee Records
🕒 Manage Attendance
📊 Dashboard Interface
🚪 Logout Functionality
🛠️ Technologies Used
Java (JDK 8 or above)
Swing & AWT (GUI)
OOP Concepts
Event Handling

⚙️ Requirements

Make sure you have:

Java JDK 8+
Any IDE (IntelliJ / Eclipse / NetBeans)

Check Java version:

java -version
▶️ How to Run the Project (Using IDE)
Clone the repository:
git clone https://github.com/your-username/EmployeeManagementSystem.git
Open project in IDE
Run any of the following files:
SplashScreen.java (Recommended)
OR LoginFrame.java
Application will start
▶️ How to Run Using Command Line
Go to project folder:
cd EmployeeManagementSystem
Compile all files:
javac *.java
Run main file:
java SplashScreen
📦 How to Create & Run JAR File
🔹 Step 1: Compile Project
javac *.java
🔹 Step 2: Create JAR File
jar cvfe EmployeeManagementSystem.jar SplashScreen *.class
🔹 Step 3: Run JAR File
java -jar EmployeeManagementSystem.jar
